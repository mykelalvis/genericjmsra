Generic Resource Adapter for JMS

J2EE 1.4 specifies connector 1.5 as the basic mechanism to integrate JMS providers with J2EE application servers. However certain JMS provider vendors do not provide a connector 1.5 compliant resource adapter to enable such an integration. This project helps such JMS providers to integrate with J2EE application servers by wrapping their JMS client library in a connector 1.5 resource adapter. We hope this resource adapter will help to connect any JMS provider with any J2EE application server such as Sun Java System Application Server.
