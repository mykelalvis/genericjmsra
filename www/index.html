Generic Resource Adapter for JMS

J2EE 1.4 specifies J2EE Connector Architecture 1.5 as the basic mechanism to integrate JMS providers with J2EE application servers. However some JMS provider vendors do not provide a J2EE Connector Architecture 1.5 compliant resource adapter to enable such an integration. This project helps such JMS providers to integrate with J2EE application servers by wrapping their JMS client library in a J2EE Connector Architecture 1.5 resource adapter. We hope this resource adapter will help to connect any JMS provider with any J2EE application server such as Sun Java System Application Server.

More Information

The technical details of the Generic Resource Adapter for JMS is available in the user guide.  It also contains the configuration steps to deploy and configure Generic Resource Adapter for JMS in J2EE 1.4 SDK or Sun Java System Application Server 8.1.

A how to guide to integrate Tibco EMS with Sun Java System Application Server using Generic Resource Adapter for JMS is available in the documents section.

A sample application that demonstrates how to integrate Tibco EMS to work with Sun Java System Application Server using the Generic Resource Adapter for JMS is available in the full bundle. Dont forget to read the how to guide before that.
